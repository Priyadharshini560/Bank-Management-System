## Bank Management System 

 Description
This project is a console-based Bank Management System developed using C++. It performs basic banking operations like account creation, deposit, withdrawal, and balance inquiry using file handling.

## Features
- Create Account
- Deposit Money
- Withdraw Money
- Check Balance
- Data stored using file handling

## Technologies Used
- C++
- Object-Oriented Programming (OOP)
- File Handling

## How to Run
1. Compile the code:
   g++ bank.cpp -o bank
2. Run the program:
   ./bank

## Output
Menu-driven console application for banking operations.
